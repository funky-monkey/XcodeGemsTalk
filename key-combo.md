Keycombos:

⌘⇧O - Quick Open File
⌘t - new tab
⌘0 / ⌘⌥0 - toggle left/right panel
⌘⌥↩ / ⌘↩ - Split Code / Single code View
⌘⇧Y - toggle console
⌘⇧J - Jump to current file
⌘⌥⇧F - Find and Replace all in Workspace.
⌘G - Cycle search results
ctrl⌘J - Look at header / implementation
⌘[ or ⌘] - Move line [up/down].
^1, ^5, ^6 -> Start typing
